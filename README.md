# Password Strength Analyzer

## Description
Password Strength Analyzer is a web application that evaluates the strength of user-entered passwords based on length, complexity, and uniqueness.

## Features
- Checks password length
- Detects uppercase and lowercase letters
- Detects numbers
- Detects special characters
- Rates password as Weak, Medium, or Strong
- Suggests improvements for weak passwords
- (Optional) Prevents reuse of old passwords using hashed storage

- Flask
- HTML
- CSS
